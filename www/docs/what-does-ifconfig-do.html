Title: What does "ifconfig" do?

The shell command "ifconfig" is named after the term interface configuration, which should be a hint as to what it is used for. I consider the ifconfig shell command as a tool used to configure interfaces. You can use the ifconfig command in a multitude of ways. Although you can use the ifconfig command to display interface information, you can also use the ifconfig command to change interface information.

For example, the command ifconfig eth0 hw ether 00:00:00:00:00:02 will change the mac address on interface "eth0."
