Title: What is a Penetration Test?

Penetration testing is an intrusive entry to a network. A penetration test is surveying, assessing, and testing the security of a given organization by using the same techniques, tactics, and tools that a malicious hacker would use. Often times, penetration tests are conducted by white hat hackers. A white hat hacker is different from a black hat hacker by the authorizations and permissions they have to exploit the vulnerabilities of a network. If a person does not have permissions or authorization to exploit vulnerabilities of a network, they're a black hat hacker. If a person has been given permission or authorization to assess and exploit vulnerabilities in a network, then they're working as a white hat hacker.
