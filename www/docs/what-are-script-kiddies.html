Title: What are Script Kiddies?

While I perceive script kiddie as a slang term, this slang term does have an actual definition. Script kiddie is a term used to describe a person who does not possess much (or any) real knowledge about hacking, but uses tools created by other developers to set fires. I am using this phrase set fires to describe the actions of setting off NIDS/IDS alarms, knocking on ports, using sophisticated open-source tools to attempt injecting malware into corporate/military server targets, etc. In this day and age, it would be difficult for a script kiddie to do much more than be of a mild annoyance. Setting a fire is the act of being mildly annoying with your network connection. Script kiddies set fires. Don't set fires. They're annoying.
