Title: Who is Localhost?

The machine you're using is defined as localhost, or the machine you're using is defined by the network address 127.0.0.1. You communicate with your own machine by using the 'localhost' address, which is 127.0.0.1. Localhost is a term used to refer to your own machine when you're on a network with other machines.
