Title: What is the CIA Triad Concept?

CIA is an acronym which stands for confidentiality, integrity, and availability. This is a conceptual model designed to help those writing information security policies within an organization. Confidentiality, integrity, and availability are the three key principles that should be used to guarantee you have a secure system.

Confidentiality: Prevents the disclosure of data to unauthorized people so that only authorized people have access to data. If unauthorized people have access to your data, then your data is insecure.

Integrity: Knowing that data has not been tampered with. If data has been tampered with, then the data is not legitimate and is useless.

Availability: Ensures that data is always available. If data isn't available for an organization to use, then the data is useless.
