Title: Qualitative Versus Quantitative Variables

Qualitative Versus Quantitative Variables

A qualitative variable represents a nonnumeric quality of an object. For example, the color of cars in a car-lot is a qualitative variable. Usually we are interested in the number or percent of observations in each category. Qualitative data are usually summarized in graphs and bar charts. Quantitative variables are usually reported numerically. They're two types of quantitative variable: continuous variables and discrete variables. Discrete variables can assume only certain values. A continuous variable can assume any value within a specified range.
