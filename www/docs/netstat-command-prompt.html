Title: Windows Command Prompt - Netstat

Use the netstat command, with the "-a" parameter, to check all active connections. A practical application of listing all active connections would be during a cyber attack where the bad actor has connected to your machine. If you're end-goal is to track the bad actor back to their own IP address, you're going to want to know about how to use the netstat -a command. For example, if the web application you host has a bad actor connected, try running the netstat -a command to collect evidence, such as IP addresses.

Use the netstat command, with the "-r" parameter, to display the computers routing tables. This is how you display the routing tables for a Windows machine -- using Windows Command Prompt. You may also do this using Windows Powershell.
